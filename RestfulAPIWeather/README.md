# RestfulAPIWeather Application
## Project Description
This is an application that uses metaweather api to show the weather for that city

## Technologies

**Android Studio**
**API-MetaWeather**
**Volley**

## Features
- call the API by ID
- call the API by city name
- call the API to name to get the Id
